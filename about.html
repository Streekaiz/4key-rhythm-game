4koneko is a simple 4 key rythm game made with pixi.js by koneko, hence the name.

I apologise for any technical faults, please report them to the github linked on my profile.

As for the songs, you can play your own custom songs (you need the .qp file) and the songs available are chosen semi randomly, so I apologise if you don't like the songs.

If you want to see extra stuff, you can type in the console (CTRL+SHIFT+I) localStorage.developerMode = "true" or localStorage.extraOptions = "true". Enjoy!
